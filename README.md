# AviUtl2-WhisperAutoSub

AviUtl2でWhisperを使用して動画・音声から自動で字幕を生成するプラグインです。

本プラグインは独自に開発されたものです。

---

# ダウンロード

右側の「Releases」から
WhisperAutoSub.aux2 をダウンロードしてください。

---

# インストール

WhisperAutoSub.aux2 を

AviUtl2の

plugins フォルダに入れてください

---

# 使い方

動画を読み込み後、本プラグインを実行すると字幕が生成されます。

---

# 必要環境

・Python 3.9以上
（動作確認：Python 3.12.10）

・FFmpeg 4.4以上
（動作確認：FFmpeg 8.0.1）

※事前にインストールし、PATHを通してください

※Whisperおよびモデルは初回実行時に自動ダウンロードされます

---

# 動作確認環境

OS：Windows 10
Python：3.12.10
FFmpeg：8.0.1
AviUtl2：最新版

---

# 注意事項

初回実行時はモデルダウンロードのため時間がかかります

---

# 使用ライブラリ

以下のオープンソースを使用しています：

・OpenAI Whisper
・faster-whisper
・FFmpeg

開発者の皆様に感謝いたします
